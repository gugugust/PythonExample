# PythonExample

본 저장소는 파이썬 프로그래밍 기초 과제 제출을 위한 Repository입니다.

## 📂 파일 목록 (File List)

1. `Python04_Condition.ipynb`
   - 파이썬 조건문(`if`, `elif`, `else`) 활용 예제
   - 논리 연산자(`and`, `or`) 및 삼항 연산자(Shorthand if) 예제 포함
2. `Python05_Function.ipynb`
   - 파이썬 함수(`def`) 정의 및 호출 예제
   - 매개변수(`args`, `kwargs`), 기본값 설정, 반환값(`return`) 예제
   - 람다(`lambda`) 함수 예제 포함

## 📖 과제 개요 (Overview)
- **주제**: 조건문(Condition) 및 함수(Function) 개념 실습
- **실행 환경**: Jupyter Notebook (Python 3)
- **목적**: 각 단원별 핵심 예제 코드(25% 이상)를 직접 작성하고 실행 결과를 확인하여 파이썬 기본 문법을 숙지합니다.
